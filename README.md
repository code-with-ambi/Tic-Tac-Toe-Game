# 🎮 Tic Tac Toe Game

A simple and interactive Tic Tac Toe game built using HTML, CSS, and JavaScript.

## 🚀 Features
- Two player gameplay
- Winner detection
- Reset & New Game button
- Responsive UI
- Clean and simple design

## 🛠️ Tech Stack
- HTML
- CSS
- JavaScript

## 📌 What I Learned

- Event Handling
- Game Logic
- JavaScript Functions
- Basic UI Design
